The Mezzanine
üîì Unlocked

The Mezzanine is located at the very top of the Studio window. Think of it as your mission control center!

What you'll find here:

  • Playtest Options: Test your game right from Studio
  • Tab Selector: Switch between Model, Avatar, Script, and UI editors
  • Collaborator Info: See who else is working on your game
  • Assistant Tool: Get AI-powered help while building

This is where you'll quickly jump between different modes and test your creations!

The Toolbar
üîí Locked

Right below the Mezzanine, you'll find the Toolbar. This is your toolbox filled with everything you need to create!

The Toolbar is organized into tabs:

  • Home: Basic building tools and commands
  • Model: Tools for creating and editing 3D objects
  • Avatar: Character and animation tools
  • UI: Create user interfaces and menus
  • Script: Coding and scripting tools
  • Plugins: Additional tools you can add

Each tab contains specific tools grouped by function - super organized!

The 3D Viewport

The 3D Viewport is the big area in the center of your screen - this is where the magic happens! It's your window into the 3D world you're creating.

You'll spend most of your time here, viewing and editing your game in 3D space. Learning to navigate this viewport is essential!

4
The Toolbox
üîí Locked

The Toolbox is your asset library - it's like a huge catalog of building materials, models, and resources!

The Toolbox has four main tabs:

  • Creator Store: Browse thousands of free models, meshes, audio, and more created by the Roblox community
  • Inventory: Access assets you own or have purchased
  • Recent: Quickly find assets you've used recently
  • Creations: Your personal creations and saved models

Pro tip: The Creator Store is perfect for finding pre-made objects to speed up your building!

5
The Explorer
üîí Locked

The Explorer shows you everything in your game organized in a tree structure. Think of it like a file explorer for your game!

Every object, service, and part in your game appears here in a hierarchical list. This makes it easy to:

  • Find specific objects quickly
  • Organize your game structure
  • See parent-child relationships between objects
  • Access game services like Workspace, Lighting, and more

The Explorer is essential for keeping your game organized as it grows more complex!

6
The Properties Panel
üîí Locked

The Properties panel is where you customize everything! When you select an object, all its properties appear here.

What can you customize?

  • Appearance: Colors, materials, transparency
  • Size & Position: Exact coordinates and dimensions
  • Behavior: Physics properties, anchoring, collision
  • Special Features: Unique properties for each object type

Want to change a part's color? Make something invisible? Adjust gravity? It all happens in the Properties panel!
