# Install Roblox Studio - Tutorial Summary

This tutorial explains how to install Roblox Studio on your Windows or Mac computer.

## About Roblox Studio

**Roblox Studio**, or **Studio** for short, is Roblox's free application with powerful 3D creation tools you can use to build and publish experiences to hundreds of millions of Roblox users on consoles, desktops, and mobile devices.

Studio comes bundled with the Roblox game engine that also powers experiences on the Roblox app. This means that when you develop and publish your experiences in Studio, players interact with the same game engine from your development on their personal devices.

## System Requirements

### Minimum Requirements
- **OS Version:** Windows 10 / macOS 10.14
- **Memory (RAM):** 3 GB

### Recommended Requirements
- **OS Version:** Windows 11 / macOS 14+
- **Memory (RAM):** 8 GB
- **Resolution:** 1600Ã—900 or higher

## Installation Steps

Follow these steps to install Roblox Studio:

### 1. Download Studio
Click the **Download Studio** button on the Roblox Creator Hub website.

### 2. Confirm Download
In the pop-up dialog, click the **Download Studio** button to confirm the download.

### 3. Locate the Installer
Find the Studio installer in your browser's download history and double-click the file.

- **On Windows:** The file is called `RobloxStudio.exe`
- **On Mac:** The file is called `RobloxStudio.dmg`

### 4. Launch Studio
After Studio finishes installing, a pop-up confirmation displays. Click the **Launch Studio** button.

### 5. Log In
After Studio opens, log in with your Roblox account.

## Next Steps

Now that you have Studio installed and open, you can explore the fundamental UI components you need to navigate Studio and start creating your first experience.
