# Explore Studio UI - Tutorial Summary

This tutorial introduces Roblox Studio's fundamental interface components for beginners.

## Key UI Sections

**Mezzanine** (Top Bar)

![Mezzanine](roblox_images/Mezzanine.jpg)

The uppermost section contains playtest options, tabs for different workflows (Model, Avatar, Script, UI, Plugins), collaborator information, and the Assistant tool.

**Toolbar**

![Toolbar](roblox_images/Toolbar.jpg)

Located below the mezzanine, it displays context-specific tools based on your active tab. The Home tab includes transform tools and part insertion; Model tab offers pivot and alignment tools; Avatar tab provides rigging utilities; UI tab contains interface object insertion; Script tab includes debugging tools.

**3D Viewport**

![3D Viewport](roblox_images/3D-Viewport.jpg)

The central workspace where you view and edit your 3D environment. After clicking in this area, you can manipulate the camera and objects.

## Camera Controls

Essential keyboard shortcuts include:
- W/A/S/D moves the camera forward/left/back/right
- Q/E moves the camera down/up
- Right mouse button dragging rotates the view
- Mouse scroll wheel zooms in/out
- F focuses on selected parts

The tutorial recommends using a 2-button mouse with scroll wheel for easier navigation.

## Supporting Windows

**Toolbox** (Left Panel)

![Toolbox](roblox_images/Toolbox.jpg)

Provides access to models, images, meshes, audio, and fonts from Roblox and community creators across four tabs: Creator Store, Inventory, Recent, and Creations.

**Explorer** (Right Panel)

![Explorer](roblox_images/Explorer.jpg)

Displays a hierarchical list of all objects and services in your current place, comparable to scenes in other engines.

**Properties** (Right Panel)

![Properties](roblox_images/Properties.jpg)

Shows customizable properties for selected objects, updating dynamically as you select different elements.
